# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

This project is in **beta**. Only the latest version on `main` receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public GitHub issue.
2. Instead, use [GitHub Security Advisories](https://github.com/oshoval/ranker/security/advisories/new) to report it privately.
3. We will work with you to understand the issue, confirm the fix, and coordinate disclosure.

Thank you for helping keep this project safe.
